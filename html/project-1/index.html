1 - Inspetor de Elementos(voltar)

O inspetor de elementos serve para depurar e analisar elementos em sua página web, podendo alterar seus aspectos e habilitam outras configurações que permitem o desenvolvedor a solucionar eventuais problemas que possam surgir em sua aplicação front-end.

2 - Estrutura Básica com HTML(voltar)

A estrutura básica do HTML é composta por tags que representam diversos elementos em uma página, como texto ou imagem, áudio e vídeo, e não apenas elementos vísiveis mas também meta informações sobre a página

3 - Falando sobre Tags(voltar)

Há diversos tipos de tags, algumas trabalham com texto, exemplo: h1 - h6, p, small, strong. Tem tags para links, imagens, audio e video, e também para scripts serem inseridos no documento. Elas representam elementos na página.

4 - Atributos das Tags(voltar)

Atributos de um elemento são usados para prover informações adicionais ao elemento, existem atributos que são comuns a todos os elementos e outros específicos, também posso criar os meus próprios usando prefixo data-*.

5 - Textos(voltar)

Nessa aula textos se referem as tags que trabalham com texto como as tags de título e subtítulo, do h1 ao h6, temos a tag p para parágrafo e várias outras que informam não apenas o tipo de texto que aquele conteúdo representa mas também fornece uma formatação.

6 - Listas Ordenadas e Não Ordenadas(voltar)

Os elementos para lista são o ul e ol, o primeiro se refere a uma lista não ordenada e o último a uma lista ordenada, os itens dessas listas são compostos por elementos li, que significam list item.

7 - Links(voltar)

Links conectam uma página a outra, podemos criar links em nossas páginas ultilizando o elemento a com o atributo href que específica o caminho onde o a página do link se encontra.
